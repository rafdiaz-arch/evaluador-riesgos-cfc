RiskCheck CFC

Objetivo del formulario

Determinar si un proyecto o iniciativa aplica para una evaluación formal de riesgos, y gestionar el proceso hasta la comunicación de conclusiones.

Criterios de aplicabilidad

Este formulario aplica para cualquier proyecto o iniciativa que involucre al menos uno de los siguientes elementos:

⚡Tratamiento de datos personales o sensibles
⚡Uso de tecnologías nuevas o innovadoras
⚡Participación de proveedores o terceros externos
⚡Cambios en procesos, sistemas o modelos de negocio
⚡Impacto en la continuidad operativa del negocio
⚡Requisitos legales o regulatorios específicos
⚡Riesgo financiero o reputacional significativo
⚡Integración con sistemas críticos del grupo

Resultados posibles

No aplica — El proyecto no requiere evaluación formal. Puede proceder.
Revisión básica — Requiere verificación rápida con el equipo de riesgos.
Evaluación formal — Debe pasar por el proceso completo de RiskCheck CFC.

Cómo completar el formulario

1
Datos del proyecto — Completa el nombre, empresa, responsable y una descripción breve del proyecto.
2
Cuestionario de 8 preguntas — Responde Sí / No / No estoy seguro para cada pregunta de aplicabilidad.
3
Resultado automático — El sistema determina el nivel de revisión requerido y activa el análisis IA.
4
Correo al equipo — Se genera un correo automático al responsable y al encargado de riesgos.
